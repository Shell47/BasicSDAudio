BasicSDAudio ChipKit Library

OVERVIEW:
BasicSDAudio is a library for the ChipKit platform that plays audio 
files with decent quality from an SD card and requires little additional
harware.

STATUS:
This is currently an alpha release to make it available for potential 
users as quickly as possible. Not all features have been tested, so
testers are welcome.

LICENSE:
This work is derived from Lutz Lisseck's SimpleSDAudio Arduino library
and is released under the same MIT License
